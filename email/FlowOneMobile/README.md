# FlowOne Pro Mobile

Native iOS (and Android) wrapper for the FlowOne Pro web app using Capacitor.

## Prerequisites

- Node.js 18+
- Xcode (from Mac App Store)
- CocoaPods (`brew install cocoapods`)
- Apple Developer account (for device testing / App Store)
- Frontend dependencies installed (`cd ../frontend && npm install`)

## Setup

```bash
npm install
npm run build:ios
```

## Development

```bash
# Build frontend + sync + open Xcode
npm run build:ios

# Or step by step:
npm run build:web       # Build the Vue frontend
npm run sync:ios        # Sync to iOS native project
npm run open:ios        # Open in Xcode

# Run on connected device or simulator
npm run dev:ios
```

## Build for App Store

1. Run `npm run build:ios` to open Xcode
2. Select your team and signing certificate
3. Product > Archive
4. Distribute App > App Store Connect

## Project Structure

```
FlowOneMobile/
  capacitor.config.ts   # Capacitor configuration
  package.json          # Dependencies and scripts
  ios/                  # Native iOS project (Xcode)
  resources/            # Icon and splash screen sources
  scripts/              # Build convenience scripts
```

## Notes

- The app wraps the same built output as the PWA (`../frontend/dist`)
- No frontend code changes are needed
- Native push notifications use APNs via @capacitor/push-notifications
- The iOS project is in `ios/` -- generated by Capacitor, committed to git
