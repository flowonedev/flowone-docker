# Assets

Place the following files in this folder:

## tray-icon.png
A 16x16 or 32x32 PNG icon for the system tray.

## icon.ico
A Windows .ico file for the application icon (recommended sizes: 16x16, 32x32, 48x48, 256x256).

## icon.png
A 512x512 PNG icon for other purposes.

---

You can create these using tools like:
- [Figma](https://figma.com)
- [Canva](https://canva.com)
- [favicon.io](https://favicon.io/favicon-converter/)
