extension Modular Addons Architecture

All features are delivered as modular addons that can be individually enabled or disabled by the admin. Each addon registers its own routes, components, sidebar entries, and API endpoints. The addon system uses a composable (useAddons.js) for runtime feature flag checks.

useAddons.js — addon feature flags
// Composable for checking addon state export function useAddons() { const isAddonEnabled = (addonSlug) => { return enabledAddons.value.includes(addonSlug) } return { isAddonEnabled, enabledAddons, addonList } }

translate Multilingual (i18n)

Full internationalization using Vue i18n with lazy-loaded locale files. Currently supports English and Hungarian with per-module translation namespaces. Locale files are organized per addon for scalability.
